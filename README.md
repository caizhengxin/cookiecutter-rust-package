# cookiecutter-rust-package

Rust package template.

## Features

- [x] clap
- [x] tracing
- [x] chrono
- [x] serde
  + [x] serde_json
  + [x] toml
- [x] thiserror
- [x] lazy_static
- [x] cargo deb

## Usage

```bash
$ pip3 install cookiecutter
$ cookiecutter https://github.com/caizhengxin/cookiecutter-rust-package.git
```
